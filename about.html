About Student Portal

The Student Portal is a simple web-based system designed to help students access academic information easily. It provides a centralized platform for managing courses, attendance, results, and personal profiles.

This project is created for learning and demonstration purposes. It shows how HTML and CSS can be used to build a clean and responsive user interface.

Key Features

  • 📚 View and manage courses
  • 📅 Track attendance records
  • 📝 Check exam results
  • 👤 Manage student profile
  • 💻 Simple and responsive design
